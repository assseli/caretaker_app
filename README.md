# Caretaker App

Caretaker App is a web platform for connecting caregivers with families and individuals in need of care services. The app allows users to manage caregivers, members, jobs, applications, and appointments all in one place.

## Features

- User-friendly web interface for managing care-related data
- Secure login and admin dashboard
- REST API for programmatic access to all core features
- Demo data population and easy setup
- Hosted live on PythonAnywhere

## Available API Endpoints

- `/api/users/`
- `/api/caregivers/`
- `/api/members/`
- `/api/addresses/`
- `/api/jobs/`
- `/api/job_applications/`
- `/api/appointments/`

## How to Use

1. Visit the live demo link above to explore the app.
2. Use the admin interface to manage users and data (admin login required).
3. Access the API endpoints to perform CRUD operations on all resources.
4. Demo scripts are available for quick testing of the API.

## About

This project is ideal for learning, prototyping, and demonstrating how modern web apps can connect people with care services. It is designed to be simple to use and easy to extend.

For more information, see the demo scripts and explore the API endpoints listed above.
